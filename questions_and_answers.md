###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>A variable named "greeting" is declared using let. Then, an empty object {} is assigned to this variable. Finally, the value of "greeting" is logged to the console, which will display an empty object {} in the console . </i>

</p>
</details>

###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>
Calling sum(1, "2") in JavaScript results in the string "12" because the second argument, "2", is a string, and JavaScript performs type coercion, treating it as a string concatenation operation rather than numeric addition.</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>The code initializes a variable `count` to 0 and an array `nums` with four numbers. It then uses the `forEach` method to iterate through `nums`, and for each non-zero number , it increments `count` by 1. Finally, it logs the value of `count`, which is 3 because there are three truthy values in the array .</i>

</p>
</details>
